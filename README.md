# Phoenix Platform

Mobile-first power systems simulator for:

- Grid ↔ Generator synchronization
- Synchroscope visualization
- Breaker logic and protection events
- Real-time system behavior simulation

## Live Demo
https://arcus13-1.github.io/phoenix-platform/

## Features
- Manual sync + auto matching
- Chaos mode (fault injection)
- Waveform visualization
- Event logging + analysis
- Mobile + tablet optimized UI

## Purpose
Built for real-world electrical testing training and system understanding.
Designed to simulate field conditions in a clean, interactive environment.

## Status
Active development — UI and system realism improvements in progress.
