When opened from iPhone Files or offline, this runs in ghost mode and still feeds the simulator. If served from Safari with internet, it attempts a public frequency fetch and quietly falls back if blocked.

Why the old builds half-worked

1. External CDN libraries made local-file behavior brittle on iPhone.
2. Some earlier builds had missing state transitions, so breaker close changed text but not physics.
3. Randomized power values looked alive but were not tied to voltage, frequency, or breaker state.
4. Files opened inside preview panes can suppress some browser behavior. Safari is safer than the document previewer.
