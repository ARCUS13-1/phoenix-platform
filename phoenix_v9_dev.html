Training Use Notes

1. This overlay provides a reference grid frequency and net-load context for the simulator. In offline mode, Phoenix uses a ghost-frequency source so training still works without internet.
2. The overlay is informational. Safe synchronizing still depends on simulator conditions: breaker status, generator readiness, ΔV, Δf, Δφ, and active faults.
3. Use this page to compare simulated machine behavior against system context, then return to the simulator tab to execute the operating sequence.
4. For best results on mobile, open the file in Safari or a full browser tab rather than a document preview pane.
